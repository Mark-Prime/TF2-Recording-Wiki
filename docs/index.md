# Overview

The goal of this page is to create an archive of all of the information known to the TF2 community on the many ways you can record and create TF2 video content.

## THIS IS A WORK IN PROGRESS

This website is still in the very very early stages and should not be expected to be perfect.
If you have anything you think should be changed, please check out the [How To Contribute](-How To Contribute/) section and add your own information to the pages.
